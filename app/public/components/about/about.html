What is that?

That is a blogging and sharing games JavaScript single-page application. It uses on a server side NodeJS, Express and MongoDB. On a client side - Navigo as a router, Handlebars as a templates compiler, SystemJS as a module loader, JQuery for ajax requests and DOM manipulations, CryptoJS for hashing, Toast for notifications. The design is supported by Twitter Bootstrap, JQueryUI and Sass.

What is that about?

The App is written as Telerik Academy JavaScript Applications course project finishing before the end of May 9th, 2017.
